

# Checklist API

This app demonstrates how I build a back-end app using Nest.js

## Features

- Create a checklist
- Update a checklist
- Delete a checklist
- List of checklist with search and/or filter

## Stacks

- Nest.js
- MongoDB
- TypeScript


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
